# Changelog

All notable changes to the Jam Kuradoberi M.U.G.E.N character.

## [1.0] - 2001-04-14

### Added
- Initial character release by AOAO
- Full Guilty Gear-style move set
- Special moves: Baku, Houchifu, Fumifumi, Kenrou, Geki, Ryujin, Asanagi
- Super moves: Choukyaku Houou Shou, Jou, Gasen
- Roman Cancel and Dead Angle Attack
- EX mode and Super mode support
- 12 color palettes

### Files
- jam.def, jam.cmd, jam.air
- Constant.cns, Jam.cns, common1.cns
- AttackNormal.cns, AttackSpecial.cns, AttackEX.cns, Attackhyper.cns
- hyper.cns, State-2.cns, State-3.cns
- Effect.cns, Extra.cns, GUARD.cns, helpers.cns
